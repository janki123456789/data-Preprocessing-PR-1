# 📊 Customer Data Analysis & Machine Learning Project

🚀 This project focuses on performing **Data Analysis, Data Cleaning, Exploratory Data Analysis (EDA), and Data Profiling** on a customer dataset.  
It also includes framing a **Machine Learning problem (Customer Churn Prediction)**.

---

## 📁 Dataset

👉 Download Dataset:

- age  
- purchase_amount  
- income  
- gender  
- churn

## 🎯 Project Objectives

- Understand data analysis fundamentals

Data Analysis is the process of collecting, cleaning, and transforming data to extract useful insights.

1. Problem Understanding  
2. Data Collection  
3. Data Cleaning  
4. EDA  
5. Feature Engineering  
6. Model Building  
7. Evaluation  
8. Deployment  

### ✔️ Machine Learning Problem:
